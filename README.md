# 💰 PocketSmart AI
### Smart Budget & Recommendation Assistant

PocketSmart AI is an **AI-powered budgeting and recommendation platform** designed to help users plan purchases and events efficiently within their budget. The application provides intelligent recommendations for **home interiors, party planning, jewelry selection, and personal finance analysis**.

The system simplifies financial decision-making by analyzing budgets and generating **smart suggestions for spending and saving**.

---

# 📌 Project Overview

Managing expenses for home decoration, events, or personal shopping can be challenging when working within a fixed budget. PocketSmart AI helps users make better financial decisions by providing:

- Smart budget allocation  
- Personalized product suggestions  
- Expense analysis  
- Intelligent financial insights  

The application provides a **modern interactive dashboard** where users can explore different budget planning modules.

---

# 🚀 Key Features

## 🏠 Home Budget Planner
The Home Budget Planner helps users allocate their budget across furniture and home décor items.

**Features**

- Budget-based recommendations  
- Room-specific planning (Living Room, Bedroom, Kitchen)  
- Smart allocation of furniture and décor items  

---

## 🎉 Party Budget Planner
This module helps users plan events efficiently within their available budget.

**Features**

- Event-based budget allocation  
- Budget split across categories such as:
  - Venue
  - Food
  - Decorations
  - Entertainment
- Visual progress indicators for spending distribution  

---

## 💎 Jewelry Budget Planner
This module suggests jewelry items based on budget and occasion.

**Features**

- Occasion-based suggestions (Wedding, Party, Casual)  
- Budget-based jewelry recommendations  
- Optional outfit image upload for personalized suggestions  

---

## 📊 Budget Analysis System
The Budget Analysis feature helps users evaluate their financial health.

**Features**

- Monthly income vs expenses analysis  
- Savings percentage calculation  
- Expense visualization using pie charts  
- Smart financial suggestions for better money management  

**Example Insights**

- Overspending detection  
- Saving improvement suggestions  
- Expense optimization tips  

---

# 🖥️ User Interface

The application uses a **tab-based dashboard interface** where users can switch between budgeting modules such as:

- Home Interiors  
- Party Planning  
- Jewelry Selection  
- Budget Analysis  

The interface is designed to provide a **simple and intuitive budgeting experience**.

# ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/yourusername/pocketsmart-ai.git
```

### Navigate to the project folder

```bash
cd pocketsmart-ai
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open the application in your browser.
